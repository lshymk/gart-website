## 🚀 开始部署（只需3步）

### 第一步：注册 GitHub 账号（如果没有）

1. 打开网站：https://github.com
2. 点击右上角 **"Sign up"**（注册）
3. 填写信息：
   - 用户名：建议用 `gart-led` 或 `gart-led-website`
   - 邮箱：您的常用邮箱
   - 密码：设置一个强密码
4. 点击 **"Create account"**
5. 验证邮箱（会收到一封邮件，点击里面的链接）

**用时：约 5 分钟**

---

### 第二步：创建 GitHub 仓库

1. 登录 GitHub 后，点击右上角的 **"＋"** 按钮
2. 选择 **"New repository"**（新建仓库）
3. 填写信息：
   - **Repository name**（仓库名）：输入 `gart-website`
   - **Description**（描述）：输入 `G-ART 谷亚LED显示屏品牌网站`
   - 选择 **"Public"**（公开）
4. 点击 **"Create repository"**（创建仓库）

**用时：约 1 分钟**

---

### 第三步：上传代码到 GitHub（最关键的一步）

**方法一：使用网站界面上传（推荐，最简单）**

1. 在刚创建的仓库页面，点击 **"uploading an existing file"** 链接
2. 把项目文件夹里的 **所有文件** 拖拽到这个页面上
3. 等待上传完成
4. 滚动到页面底部，在 **"Commit changes"** 框里输入：
   - 标题：`上传网站代码`
5. 点击绿色按钮 **"Commit changes"**

**用时：约 2-3 分钟**

**方法二：使用 Git 命令（如果您会使用命令行）**

```bash
# 在项目文件夹打开命令行
git remote add origin https://github.com/您的用户名/gart-website.git
git branch -M main
git push -u origin main
```

---

### 第四步：部署到 Vercel（获得免费域名）

1. 打开网站：https://vercel.com
2. 点击右上角 **"Sign Up"**（注册）
3. 选择 **"Continue with GitHub"**（使用GitHub登录）
4. 授权 GitHub 访问
5. 登录后，点击 **"Add New..."** → **"Project"**
6. 找到刚才创建的 `gart-website` 仓库，点击 **"Import"**（导入）
7. 配置项目（不用改任何东西，直接默认即可）：
   - **Framework Preset**：选择 `Next.js`
   - **Root Directory**：保持为 `./`
8. 点击 **"Deploy"**（部署）
9. 等待 1-2 分钟，网站就会部署完成！
10. 部署成功后，您会看到一个链接，比如：
    - `https://gart-website.vercel.app`

**用时：约 3-5 分钟**
