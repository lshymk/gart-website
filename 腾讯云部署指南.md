# 腾讯云静态网站托管部署指南

## 📋 部署准备

### 1. 已完成的工作
- ✅ 静态文件已构建（位置：`/workspace/projects/out/`）
- ✅ 上传脚本已准备（位置：`/workspace/projects/scripts/upload-tencent.sh`）
- ✅ CLI工具已准备

### 2. 部署总览
- 预计时间：10-15分钟
- 难度：简单
- 费用：免费（10GB存储 + 10GB流量/月）

---

## 🚀 部署步骤

### 第一步：注册腾讯云账号

1. 访问：https://cloud.tencent.com/
2. 点击"免费注册"
3. 填写信息并完成实名认证

---

### 第二步：创建静态网站托管环境

1. 登录腾讯云控制台
2. 搜索"静态网站托管"
3. 进入：https://console.cloud.tencent.com/tcb/hosting
4. 点击"新建应用"
5. 选择"静态网站托管"
6. 填写信息：
   - **应用名称**：G-ART谷亚官网
   - **环境**：选择新环境或已有环境
7. 点击"创建"
8. **记住环境ID**（格式类似：`g-art-website-xxxxx`）

---

### 第三步：安装腾讯云CLI工具

**方法1：自动安装（推荐）**
```bash
# 运行上传脚本时会自动安装
cd /workspace/projects
./scripts/upload-tencent.sh
```

**方法2：手动安装**
```bash
npm install -g @cloudbase/cli
```

---

### 第四步：登录腾讯云CLI

运行以下命令：
```bash
cloudbase login
```

**登录方式**：
1. 扫描二维码登录（推荐）
2. 或使用密钥登录

---

### 第五步：上传文件

运行上传脚本：
```bash
cd /workspace/projects
./scripts/upload-tencent.sh
```

**脚本会执行**：
1. 检查CLI工具是否安装
2. 检查是否已登录
3. 列出可用环境
4. 询问环境ID
5. 上传 `out/` 目录的所有文件

**输入环境ID**：
- 输入第二步中记住的环境ID
- 例如：`g-art-website-xxxxx`

---

### 第六步：访问网站

上传成功后，访问地址为：
```
https://你的环境ID.service.tcloudbase.com
```

例如：
```
https://g-art-website-xxxxx.service.tcloudbase.com
```

---

## 🔧 高级配置（可选）

### 配置自定义域名

1. 进入腾讯云控制台 → 静态网站托管
2. 点击你的应用
3. 点击"域名设置"
4. 点击"添加域名"
5. 输入你的域名（例如：`www.g-art.com`）
6. 按照提示配置DNS解析

### 配置HTTPS证书

1. 进入"域名设置"
2. 点击你的域名
3. 点击"HTTPS配置"
4. 选择"自动免费证书"（如果有）
5. 或上传自己的证书

### 设置用量告警

1. 进入腾讯云控制台 → 消息中心
2. 点击"用量告警"
3. 选择"静态网站托管"
4. 设置告警规则：
   - 存储使用量达到8GB时通知
   - 流量使用量达到8GB时通知

## 🔄 更新网站

### 修改代码后重新部署

1. 修改代码
2. 重新构建：
   ```bash
   cd /workspace/projects
   pnpm run build
   ```
3. 重新上传：
   ```bash
   ./scripts/upload-tencent.sh
   ```

### 仅更新部分文件

如果只修改了少量文件，可以：

**方法1：网页上传**
1. 进入静态网站托管控制台
2. 点击"文件管理"
3. 上传修改的文件

**方法2：CLI上传**
```bash
# 上传单个文件
cloudbase hosting deploy out/index.html -e 环境ID

# 上传整个目录
cloudbase hosting deploy out -e 环境ID
```

## 🐛 常见问题

### Q1：上传脚本报错"command not found"
**解决**：
```bash
# 确保脚本有执行权限
chmod +x /workspace/projects/scripts/upload-tencent.sh

# 确保在正确的目录
cd /workspace/projects
./scripts/upload-tencent.sh
```

### Q2：上传速度慢
**原因**：文件较多或网络慢
**解决**：
- 等待上传完成
- 或使用网页上传部分文件

### Q3：访问网站404
**原因**：文件上传不完整或路径错误
**解决**：
1. 检查 `out/index.html` 是否存在
2. 重新上传所有文件
3. 检查环境ID是否正确

### Q4：无法登录腾讯云CLI
**解决**：
1. 检查网络连接
2. 确认腾讯云账号密码正确
3. 尝试使用密钥登录
